# HYP-002: Two-layer vs single-layer error propagation

## Claim
Splitting the pipeline into layers **A** (upstream) and **B** (downstream) reduces end-to-end error propagation by at least **Z%** versus a single-layer design **S**, under matched total work / same formula depth.

## Propagation metric
```text
Δ = (e_S − e_AB) / e_S × 100%    (want Δ ≥ Z%)
```

## Falsify criteria
- **REJECT** if Δ < Z% on agreed benchmark after ≥ 3 seeds / splits.
- **REJECT** if two-layer only wins by adding hidden capacity (unmatched work vs S).
- **PAUSE** if A, B, G are undefined — must pin minimal toy implementation first.
